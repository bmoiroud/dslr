# DSLR

## Description

Le but de ce projet est de découvrir le nettoyage de données et la régression logistique.

## Installation

``` bash
$> pip install -r ./requirements.txt
```

## Utilisation

### Entrainement

```bash
$> py ./logreg_train.py <dataset.csv>
```

### Prédiction

```bash
$> py ./logreg_predict.py <dataset.csv>
```
